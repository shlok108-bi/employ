About This Project

This project aims to predict employee productivity in a garment manufacturing environment using machine learning. The model is trained on the Garments Worker Productivity dataset, utilizing features such as team, targeted productivity, standard minute value (SMV), overtime, and more to predict actual productivity.

The application uses a Flask backend with an XGBoost model for predictions. Users can input data through a web interface to receive productivity predictions. The project includes data preprocessing, model training, and a user-friendly UI.

Built as a demonstration of machine learning and web development skills, this project showcases end-to-end development from data analysis to deployment.
